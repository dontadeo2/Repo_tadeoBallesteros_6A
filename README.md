## Descripción
Este proyecto consiste en la creación de una API REST utilizando Node.js, Express y MongoDB, aplicando buenas prácticas de organización en capas (routes, controllers, models, db).
Se implementaron operaciones CRUD completas para gestionar datos almacenados en MongoDB Atlas.


## Tecnologías utilizadas
- Node.js
- Express
- MongoDB Atlas
- Mongoose
- Nodemon
- Dotenv
